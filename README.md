# 🍽️ Lavos Catering System

**A complete static website for a professional catering business** – featuring a modern landing page, service showcase, sample menu, contact form, and interactive notifications. Built with HTML, CSS, and JavaScript, and ready to deploy on any static hosting service (Netlify, GitHub Pages, Vercel, etc.).

![Lavos Catering Preview](https://images.pexels.com/photos/262978/pexels-photo-262978.jpeg?auto=compress&cs=tinysrgb&w=600)

## ✨ Features

- **Hero section** with call‑to‑action button
- **Service cards** (Weddings, Corporate Events, Private Parties)
- **Sample menu gallery** with pricing
- **Client testimonial** design
- **Contact & quote request form** (works with Netlify Forms out of the box)
- **Newsletter subscription** mock
- **Floating toast notifications** for user actions (form submission, validation errors, newsletter signup)
- **Fully responsive** (mobile‑friendly using Bootstrap 5)
- **Smooth scrolling** and anchor links
- **Back to top** button
- **No backend required** – can be extended with Netlify Forms, Formspree, or any form service
